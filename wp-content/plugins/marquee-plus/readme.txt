== Installation ==

This section describes how to install the plugin and get it working.

1. Download the plugin.
2. Extract It.
3. Upload `marquee-plus.php` to the `/wp-content/plugins/` directory.
4. Activate the plugin through the 'Plugins' menu in WordPress.
5. Options can be edited from 'Settings/Marquee-Plus'.

If any problem persists, we request you kindly to contact us and tell the problem so that we can rectify our plugin accordingly. If you are a programmer and interested in changing/rectifying the problem, you are welcomed.
