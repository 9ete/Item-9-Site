NextGEN Monoslideshow is a addon plugin for NextGEN Gallery plugin and the Monoslideshow flash slideshow

== Description ==

NextGEN Monoslideshow is a addon plugin for <a href="http://wordpress.org/extend/plugins/nextgen-gallery/" title="NextGEN Gallery">NextGEN Gallery</a> plugin and <a href="http://www.monoslideshow.com/" title="Monoslideshow by Monokai">Monoslideshow</a> flash slideshow. 

Monoslideshow is a Flash slideshow to view your images and videos in a web page. It can be placed on every website, in whatever size you want and filled with whatever content you like. It's designed for maximum flexibility, 
whether you want to have a professional looking portfolio, or a simple image rotator on your website.

<a href="http://nextgen-gallery.com/slideshow/nextgen-monoslideshow/" title="NextgEN Monoslideshow">See here for a Demo</a> 

**You need to have a running NextGEN Gallery installation to use this plugin. Please note that you need to buy Monoslideshow, it's not included in this plugin.**

= Features =

* Easy setup : Add the slideshow with just a simple shortcode
* Preset : More than 10 stellar transitions (Ken Burns, 3D, Zoom, etc)
* Integration : Replace the default slideshow with just one click

== Installation ==

1. 	Upload the <a href="http://www.monoslideshow.com/" title="Monoslideshow by Monokai">Monoslideshow.swf</a> flash file to your WordPress Upload folder

2.  Setup the correct path in the Monoslideshow options

3. 	Activate the plugin

4.	Add a gallery and upload some images

5. 	Go to your post/page an enter the tag '[monoslideshow id=x]'.

That's it ... Have fun
