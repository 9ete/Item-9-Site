== Description ==

The Twitter widget...widget plugin allows you to place a Twitter Profile widget in the sidebar. You can configure the Twitter user, size and colors. Future releases will add further configurable options as per Twitter's profile widget generator.

== Installation ==

1. Download the zipped folder
2. Unzip it
3. Copy the twitterwidget.php file into your plugins folder 
== Frequently Asked Questions ==

= Why use Twitter Widget...Widget =

This is one of the most configurable twitter widgets out there.
